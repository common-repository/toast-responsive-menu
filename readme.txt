A super lightweight mobile menu with awesome customisable features. This plugin solves a problem commonly flagged on Google PageSpeed insights. We've packaged the bare minimum you need to get a responsive menu setup on your website so you can focus on the more important stuff.

== Description == 
The Toast Mobile menu is a highly customisable and user friendly solution to provide a mobile menu on your WordPress website.

Our developers have created an easy to use both frontend & backend menu plugin with one thing in mind... SITE SPEED.

Google's new algorithm ranks websites on a mobile first basis now. If you haven't read up on it, all you need to know is it's very important to offer an awesome user experience on mobile which sometimes can be hard without the right tools.

As we're developers ourselves, we have been constantly striving to do this on our clients websites which is why we created this plugin.

<h3>Key features</h3>
<ul>
<li><strong>LIGHTWEIGHT:</strong> No external libraries or frameworks slowing down the frontend of your site</li>
<li><strong>Easy to use:</strong> An easy to use no nonsense backend interface.</li>
<li><strong>Awesome features:</strong> A large amount of customisable features straight out of the box.</li>
<li><strong>Fast setup:</strong> Set up a menu in a matter of seconds</li>
<li><strong>Tailor to your brand:</strong> A high collection of customisable options including colours and logo placement.</li>
<li><strong>Hooks:</strong> Hooks are in place within the menu so you can add additional features.</li>
</ul>
